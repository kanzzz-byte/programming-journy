CSS (Cascading Style Sheets): Pengatur Tampilan Website

CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mengatur tampilan dan tata letak halaman web. Dengan CSS, pengembang dapat mengubah warna, jenis huruf, ukuran teks, jarak antar elemen, hingga membuat animasi sederhana pada website. CSS bekerja bersama HTML, di mana HTML berfungsi sebagai struktur halaman, sedangkan CSS bertugas memperindah tampilannya. Penggunaan CSS membuat website terlihat lebih menarik, konsisten, dan responsif di berbagai perangkat seperti komputer, tablet, dan smartphone.
