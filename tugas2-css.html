Internal CSS: Mengatur Gaya Website dalam Satu Halaman

Internal CSS merupakan teknik penulisan CSS yang disimpan di dalam dokumen HTML yang sama. Kode CSS ini ditulis di dalam tag style pada bagian head sehingga hanya berlaku untuk satu halaman web saja. Penggunaan Internal CSS memudahkan pengembang dalam mengatur desain halaman secara terpusat tanpa harus membuat file CSS terpisah. Cara ini sering digunakan untuk latihan, proyek kecil, atau ketika ingin menerapkan gaya khusus pada halaman tertentu.
